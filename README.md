# penguin_checker

## 概要
機械学習を利用して、ペンギンの画像からペンギンの種類を判別するWebアプリケーション
以下の5種類から判別する
- adelie penguin(アデリーペンギン)
- emperor penguin(コウテイペンギン、エンペラーペンギン)
- jackass penguin(ケープペンギン、ジャッカスペンギン、アフリカペンギン)
- king penguin(オウサマペンギン、キングペンギン)
- rockhopper penguin(イワトビペンギン)

## 使用技術等
- scikit-learn
- openCV
- Pillow
- virtualenv
- Flask
- and more...
